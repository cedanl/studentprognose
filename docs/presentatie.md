# Presentatie

Een introductie tot **Studentprognose**, gegeven op **HOlink 2026**. De presentatie legt
in begrijpelijke taal uit waarom instellingen jaarlijks worstelen met het inschatten van
hun eerstejaarsinstroom, hoe Studentprognose werkt (van data via validatie en modellen
naar een ensemble-voorspelling) en hoe je er zelf mee aan de slag gaat.
